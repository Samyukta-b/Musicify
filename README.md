# Musicify

## Purpose
Musicify is a web application that allows users to generate custom Spotify playlists using AI. Users provide their Spotify User ID, a theme (e.g., "upbeat pop for a road trip"), and the number of songs desired (1-10). The app leverages OpenAI's GPT-3.5-turbo-instruct to create a playlist name, description, and curated song list, along with album-inspired artwork generated by DALL-E.

The application then searches Spotify for the recommended songs, creates a new private playlist in the user's account, and embeds it directly in the interface for immediate playback.

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript (with jQuery for DOM manipulation)
- **Backend/APIs**:
  - OpenAI API GPT-3.5-turbo-instruct (text generation)
  - DALL-E (image generation)
  - Spotify Web API (searching tracks, creating playlists, and embedding)
- **Dependencies**:
  - Express.js (Node.js framework, server-side handling)
  - jpeg-js (JPEG image processing)
  - pica (image resizing and manipulation)

## How to Run
1. Clone the repository.
2. Install dependencies: `npm install`
3. Set up your OpenAI API key in `env.js` (replace the placeholder).
4. Start the server: `npm start` (assuming Express is configured).
5. Open `index.html` in a browser or serve it.

Note: Ensure you have valid API keys and Spotify authentication set up.

## Features
- AI-powered playlist generation based on user themes.
- Custom artwork creation.
- Direct Spotify integration for playlist creation and embedding.
- Responsive UI with loading indicators and error handling.
